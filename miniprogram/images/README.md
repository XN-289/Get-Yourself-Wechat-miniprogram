# Tab 图标说明

需要 8 个 png 文件（81x81px，透明背景）：

| 文件名 | 用途 |
| --- | --- |
| tab-discover.png | 发现 tab（灰色） |
| tab-discover-active.png | 发现 tab（紫色 #4F46E5） |
| tab-challenge.png | 挑战 tab（灰色） |
| tab-challenge-active.png | 挑战 tab（紫色） |
| tab-achievement.png | 成就 tab（灰色） |
| tab-achievement-active.png | 成就 tab（紫色） |
| tab-profile.png | 我的 tab（灰色） |
| tab-profile-active.png | 我的 tab（紫色） |

## 快速获取方式

1. **iconfont.cn** — 搜索"发现""挑战""成就""用户"，下载 png
2. **flaticon.com** — 搜索 discover, challenge, trophy, user
3. **自己画** — 用 Figma/Sketch 画简单图标

## 临时方案

开发阶段可以用纯色方块替代，不影响功能测试。
