# Get Yourself 微信小程序测试计划

## 一、测试环境

### 1.1 前端
- 微信开发者工具（最新稳定版）
- 真机调试（iOS / Android 各一台）

### 1.2 后端
- 本地运行 Spring Boot（localhost:8080）
- MySQL 本地实例
- Redis 本地实例

### 1.3 微信配置
- 小程序 AppID（需要注册）
- 微信开发者工具中配置合法域名（或勾选"不校验合法域名"用于开发）

## 二、功能测试用例

| 编号 | 用例 | 步骤 | 预期结果 |
|------|------|------|----------|

## 七、已知限制（MVP）

1. 手机号绑定需要微信后台配置 getPhoneNumber 权限
2. 订阅消息需要微信后台配置模板
3. AI推荐依赖后端AI服务配置
4. tab图标需要替换为实际图片
