# 快速测试指南

## 前置条件

- 微信开发者工具已安装
- MySQL 已启动
- Redis 已启动

## 步骤一：启动后端

```bash
# 1. 进入后端目录
cd D:\Get-Yourself\backend

# 2. 创建数据库（如果还没有）
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS getyourself CHARACTER SET utf8mb4;"

# 3. 设置环境变量
set WECHAT_APPID=test_appid
set WECHAT_SECRET=test_secret

# 4. 启动
mvn spring-boot:run
```

启动成功后会看到：
```
Started GetYourselfApplication in X.XX seconds
```

## 步骤二：导入小程序

1. 打开微信开发者工具
2. 选择"导入项目"
3. 目录选择 `D:\Get-Yourself-mini\miniprogram`
4. AppID 选择"测试号"或填入你的真实AppID
5. 点击"编译"

## 步骤三：测试核心流程

### 3.1 登录测试

1. 打开小程序
2. 自动触发登录（或点击"微信一键登录"）
3. 检查控制台是否有 `[Auth] 登录成功` 日志

**预期结果：** 进入首页，显示事件列表

### 3.2 事件浏览测试

1. 查看首页事件列表
2. 点击分类标签筛选
3. 搜索关键词

**预期结果：** 显示种子数据中的10个事件

### 3.3 事件详情测试

1. 点击任意事件卡片
2. 查看详情信息
3. 点击"预约参加"

**预期结果：** 预约成功，按钮变为"标记完成"

### 3.4 挑战测试

1. 切换到"挑战"tab
2. 点击"+"创建挑战
3. 填写信息并提交
4. 点击挑战 → 标记完成 → 填写反思

**预期结果：** 挑战状态变为"已完成"

### 3.5 成就测试

1. 切换到"成就"tab
2. 查看完成记录
3. 点击记录 → 编辑反思

**预期结果：** 显示完成的事件和挑战记录

### 3.6 AI推荐测试

1. 首页点击"AI智能推荐"
2. 输入需求如"想找实习机会"
3. 点击"获取推荐"

**预期结果：**
- 如果配置了AI_KEY：返回AI推荐结果
- 如果没有配置：返回规则推荐结果

## 常见问题排查

### 登录失败

**现象：** 点击登录无反应或报错

**检查：**
1. 后端是否启动
2. 控制台是否有错误日志
3. `app.js` 中 `baseUrl` 是否正确

### 事件列表为空

**现象：** 首页显示"暂无活动"

**检查：**
1. 数据库是否有数据
2. Flyway 迁移是否执行
3. 后端日志是否有SQL错误

### 网络请求失败

**现象：** 提示"网络异常"

**检查：**
1. 微信开发者工具 → 详情 → 本地设置 → 勾选"不校验合法域名"
2. 后端是否在 8080 端口运行
3. 防火墙是否阻止连接
